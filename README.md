# 🎓 Student Portfolio & Academic Management Website

A responsive multi-page web application developed as part of the **COS 106 – Introduction to Web Technologies** term project. The website serves as a personal academic portfolio and student management platform, showcasing a student's profile, skills, academic achievements, projects, and providing an interactive academic planner.

## 📖 Project Overview

This project demonstrates the application of modern web development technologies, including **HTML5**, **CSS3**, and **JavaScript**, to create a responsive and user-friendly website. It features multiple interconnected pages, interactive elements, and client-side validation while adhering to web development best practices.

## ✨ Features

- 🏠 **Homepage**
  - Student profile and photograph
  - Welcome message
  - Navigation menu
  - Brief biography

- 👤 **About Me**
  - Educational background
  - Career aspirations
  - Technical skills
  - Hobbies and interests

- 💼 **Projects**
  - Showcase of three sample projects
  - Project descriptions
  - Images/screenshots
  - Project links

- 📅 **Academic Planner**
  - Add new tasks
  - Mark tasks as completed
  - Delete tasks
  - Dynamic task management using JavaScript

- 📧 **Contact**
  - Contact form
  - JavaScript form validation
  - Email validation
  - Phone number validation

## 🛠️ Technologies Used
